🗺️ CLICKABLE PROVINCES FEATURE - READY! 🗺️

Your Canada Data Explorer now has INTERACTIVE PROVINCES!

✨ WHAT'S NEW:

✅ All 13 provinces & territories are clickable
✅ Hover to highlight province areas (indigo glow)
✅ Click to zoom smoothly to province bounds
✅ See cities available in each province
✅ Two-level navigation: Province → City

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎮 HOW TO USE:

1. Global View (default)
   → All provinces visible with subtle outlines

2. Hover Over Province
   → Area highlights in bright indigo
   → Cursor becomes a pointer
   → Ready to click!

3. Click Province
   → Map zooms to province bounds
   → City info panel shows: "📍 [Province Name]"
   → City markers become more visible

4. Click City
   → Shows detailed data & demographics
   → See charts and statistics
   → Option to enter street view

🎨 VISUAL DESIGN:

Provinces are shown as indigo areas:
  • Rest state: 3% opacity (barely visible)
  • Hover state: 15% opacity (bright highlight)
  • Border: Thin indigo outline (thickens on hover)
  • Color: Matches app's primary indigo theme

🛠️ TECHNICAL:

Data Structure (data.js):
  - CANADIAN_PROVINCES: Array of all provinces
  - getProvinceById(id): Lookup province
  - getCitiesByProvince(name): Get cities in province

Map Logic (app.js):
  - addProvinceLayer(): Creates map layers
  - selectProvince(): Handles province clicks
  - GeoJSON: Rectangle boundaries for each province

UI Updates (index.html):
  - Legend mentions "Click Provinces"
  - Tooltip includes province selection
  - Info panel shows province name when selected
